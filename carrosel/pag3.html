Instalação do Servidor Web

Nesta página, vamos explicar passo a passo como instalar um servidor web em seu sistema operacional. Um servidor web é responsável por hospedar e servir páginas e aplicativos web.

Passo 1: Atualização do Sistema Operacional

// atualização do SO
sudo apt update
sudo apt upgrade

Antes de iniciar a instalação do servidor web, é importante garantir que seu sistema operacional esteja atualizado. O comando "sudo apt update" atualiza a lista de pacotes disponíveis, enquanto o comando "sudo apt upgrade" atualiza os pacotes instalados em seu sistema operacional.

Passo 2: Instalação do Servidor Apache

// instalação do Apache 
sudo apt install apache2 apache2-doc apache2-utils
sudo systemctl status apache2

O servidor web Apache é uma das opções mais populares e amplamente utilizadas. O comando "sudo apt install apache2 apache2-doc apache2-utils" instala o servidor web Apache e seus utilitários relacionados. Uma vez concluída a instalação, você pode verificar o status do Apache com o comando "sudo systemctl status apache2".

Passo 3: Configuração do Firewall

// instalação do firewall
sudo apt install ufw
sudo systemctl status ufw

Para garantir a segurança do servidor web, é recomendável configurar um firewall. O comando "sudo apt install ufw" instala o Uncomplicated Firewall (UFW) em seu sistema operacional. Uma vez instalado, você pode verificar o status do UFW com o comando "sudo systemctl status ufw".

Passo 4: Liberação do Apache no Firewall

// liberação do Apache no Firewall
sudo ufw allow 'Apache Full'

Agora, é necessário permitir que o tráfego HTTP e HTTPS chegue ao servidor Apache através do firewall. O comando "sudo ufw allow 'Apache Full'" abre as portas necessárias (80 para HTTP e 443 para HTTPS) no firewall para permitir o acesso ao servidor Apache.

Conclusão

Parabéns! Você concluiu com sucesso a instalação do servidor web Apache e configurou o firewall para permitir o acesso a ele. Agora você pode começar a hospedar e servir páginas e aplicativos web em seu servidor.
